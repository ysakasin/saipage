# Saipage

[![Build Status](https://travis-ci.org/ysakasin/saipage.svg?branch=master)](https://travis-ci.org/ysakasin/saipage)

A diceroll tool for TRPG


## Official site

https://ysakasin.github.io/saipage/


## Build Requires

- Node (>=10.0.0)


## Build

``` bash
# install dependencies
npm install

# build client
npm build
```

## Run dev-server

```
npm start
```
